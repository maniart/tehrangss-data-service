Tehran Inside Out

This summer from July 3rd to 17th, Tehran hosts its 2nd Global Summer School , an intensive two week workshop focusing on revealing hidden aspects of urban landscapes through real time data collection, analysis, visualization, prototyping and fabrication. Combining hands-on workshops and theoretical lectures, participants will work together to use DYI tools to collect emotion data from citizens and create immersive installations and interventions that bring practices of design, architecture, fabrication & programing together to propose platforms that activate and reshape urban landscapes.

This site serves as an entry point and documentation for the web service backing data collection, parsing and distribution for this workshop. Using these endpoints, the participants will be able to collect and access realtime emotion data from Tehran, and use it to prototype and build their data-driven installations.

Endpoints

  • [GET] /emotions
    params: count[INT]
    Returns a json object with the size specified via the `count` parameter. The object contains `timestamp`, `location` and one or more emotion vectores.
  • [POST] /emotion
    Use this endpoint to post an emotion record to the service. The request body should adhere to the following schema { location: { lon: FLOAT lat: FLOAT }, emotions: { anger: FLOAT, fear: FLOAT, happiness: FLOAT, sadness: FLOAT, surprise: FLOAT, disgust: FLOAT } }
